# ChocoBerry Gift Assistant

Мобильное веб-приложение для кондитерской ChocoBerry. Главный сценарий не каталог, а подбор подарка через короткий квиз.

## Запуск

```bash
npm install
npm run dev
```

Локально приложение открывается на `http://127.0.0.1:5173`.

## Запуск с общей админкой

Для режима, где изменения из админки видят все клиенты, запускайте сервер:

```bash
npm run start
```

По умолчанию сервер откроется на `http://localhost:8080`.

Пароль админки задается переменной окружения `ADMIN_PASSWORD`. Если ее не указать, используется `choco9380`.

```bash
ADMIN_PASSWORD=your-password npm run start
```

Админка находится по адресу `/#admin`.

## Деплой

Для постоянного хостинга добавлены `Dockerfile` и `render.yaml`. На Render можно создать Web Service из репозитория, задать `ADMIN_PASSWORD`, и приложение будет работать как один сервер: публичная витрина на `/`, админка на `/#admin`.

## Данные

Исходные данные лежат отдельно от кода:

- `src/data/products.json` - товары, цены, фото, состав, категории и теги для подбора. Основное фото товара меняется в поле `image`. Для нескольких фото можно добавить поле `images`:
- `src/data/contacts.json` - адрес, WhatsApp, Instagram и график работы.

```json
"image": "/photos/product-main.jpg",
"images": ["/photos/product-main.jpg", "/photos/product-2.jpg"]
```

- `src/data/reviews.json` - отзывы клиентов.
- `src/data/gallery.json` - фотографии для раздела "Наши работы". Сейчас файл пустой, поэтому приложение показывает "Работы скоро появятся". Добавьте объект и меняйте путь к фото в поле `image`:

```json
{
  "id": "gallery-new",
  "title": "Подарочный набор",
  "image": "/photos/gallery-new.jpg",
  "height": "medium"
}
```

Встроенная админ-панель доступна по скрытому адресу `/#admin`. Она позволяет добавлять, редактировать и удалять товары, менять цены, загружать фотографии и добавлять отзывы без изменения кода.

В серверном режиме админка сохраняет изменения в `server/data/*.json`, а загруженные фотографии кладет в `public/uploads`. Эти изменения видят все клиенты, открывающие приложение с этого сервера.

Отзывы можно добавить через `src/data/reviews.json` или через админ-панель:

```json
{
  "id": "review-new",
  "name": "Имя клиента",
  "role": "Повод или заказ",
  "text": "Текст отзыва",
  "rating": 5
}
```
